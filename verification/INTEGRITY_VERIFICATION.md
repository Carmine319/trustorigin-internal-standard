# Integrity Verification Model

## Core Mechanism

Artefacts are integrity-anchored via deterministic hashing.

Verification confirms:

- Structural consistency  
- Lineage integrity  
- Export survivability  
- Freeze state preservation  

---

## Verification Scope

Integrity verification does not:

- Certify compliance  
- Validate content accuracy  
- Adjudicate ownership  

It confirms structural continuity only.
